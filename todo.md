# План замены курсора

## Цель
Заменить сложный SVG курсор на более простой и тематически подходящий для цветочного магазина

## Задачи
- [ ] Анализ текущего курсора в CSS
- [ ] Выбор простого и тематического курсора (pointer с листиком или простой зеленый круг)
- [ ] Замена курсора в основных CSS правилах
- [ ] Проверка работы курсора на всех интерактивных элементах
- [ ] Тестирование в браузере

## Варианты нового курсора
1. Стандартный pointer (стрелка с указательным пальцем)
2. Простой зеленый круг
3. SVG листик
4. Комбинация: pointer + листик в углу
