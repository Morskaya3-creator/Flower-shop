# План реализации якорных ссылок с переключением табов

## Задача
Реализовать функционал, при котором клик на элемент списка в .catalog (футер) выполняет плавный скролл к секции .our-plants и активирует соответствующий таб.

## План выполнения
- [ ] 1. Модифицировать HTML - добавить data-атрибуты к ссылкам каталога в футере
- [ ] 2. Добавить JavaScript обработчик для кликов по ссылкам каталога
- [ ] 3. Реализовать плавный скролл к секции растений
- [ ] 4. Добавить программное переключение табов
- [ ] 5. Интегрировать с существующей системой табов
- [ ] 6. Протестировать функционал

## Детали реализации
- Связать ссылки с табами через data-category атрибуты
- Учесть высоту фиксированного хедера при скролле (70px)
- Сохранить существующую функциональность прямого клика по табам
